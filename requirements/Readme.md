# Requisitos

Los requisitos de la ontología serán incluidos una vez que se generen, de manera semiautomática, preguntas de competencia a partir de la ontología que se ha generado.

En principio, la tarea de generación de preguntas de competencia se realiza durante las fases iniciales del desarrollo de las ontologías, pero en el caso de esta ontología, se ha tenido que partir de la modelización ya realizada y su implementación en OWL.
